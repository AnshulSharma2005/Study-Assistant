# 📚 AI Study Assistant

An AI-powered Study Assistant that converts notes into interactive flashcards using OpenRouter AI. This project helps students revise concepts quickly by generating question-answer flashcards from any study material.

---

## ✨ Features

- 🤖 AI-powered flashcard generation
- 📝 Paste notes and generate flashcards instantly
- 🎯 Select the number of flashcards
- 📊 Choose difficulty level (Easy / Medium / Hard)
- 🔄 Interactive flip cards
- 📋 Copy all flashcards
- 📄 Download flashcards as PDF
- 📱 Responsive design
- ⚡ Fast React + Vite frontend
- 🌐 Node.js + Express backend
- 🔗 OpenRouter AI Integration

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/AnshulSharma2005/Study-Assistant.git
```

Go to project folder

```bash
cd AI-Study-Assistant
```

---

## Frontend Setup

```bash
cd client

npm install

npm run dev
```

Runs on

```
http://localhost:5173
```

---

## Backend Setup

Open another terminal

```bash
cd server

npm install

npm start
```

Runs on

```
http://localhost:5000
```

---

## 🔑 Environment Variables

Create a `.env` file inside the **server** folder.

```env
OPENROUTER_API_KEY=your_api_key_here
```
